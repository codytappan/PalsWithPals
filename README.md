# PalsWithPals

Cost-optimized, on-demand Palworld dedicated server on AWS, controlled from Discord.

This repository will contain Terraform, an AWS Lambda Discord bot, and EC2 provisioning
scripts to run [`thijsvanloef/palworld-server-docker`](https://github.com/thijsvanloef/palworld-server-docker)
on demand — starting/stopping the server from Discord slash commands and automatically
shutting down when no players are online.
